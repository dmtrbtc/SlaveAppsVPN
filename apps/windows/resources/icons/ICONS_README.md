# Icons

Place the following icon files here:

- `icon.ico` — Windows app icon (256x256, multi-resolution ICO)
- `icon.png` — PNG version (512x512)
- `tray/icon.png` — Tray icon disconnected state (16x16 or 32x32)
- `tray/icon-connected.png` — Tray icon connected state (16x16 or 32x32)

Tray icons should be monochrome/white for proper rendering on dark/light taskbars.
Use transparent background.
